# LED Library

Uma biblioteca pública para criação de um objeto LED, que contém funções nao-bloqueaveis. A biblioteca é feita para ser utilizada em microcontroladores, como o ESP32, escrita em C++. 

## Features

- Acender o LED (Com opção de definir a duração da ação).
- Desligar o LED.
- Alternar o estado do LED.
- Piscar o LED (Com opção de alterar a frequencia do LED e a quantidade de vezes em que pisca).
